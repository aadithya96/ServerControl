Legacy PNG launcher icons (required for Android < API 26).
Generate these from ic_launcher_foreground.xml using Android Studio:
  Right-click res/ → New → Image Asset → Launcher Icons (Adaptive and Legacy)

Sizes:
  mipmap-mdpi:    48x48
  mipmap-hdpi:    72x72
  mipmap-xhdpi:   96x96
  mipmap-xxhdpi:  144x144
  mipmap-xxxhdpi: 192x192
